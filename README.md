# 李白诗词 · 诗仙遗韵

> 唯美的李白诗词展示网站，随机展示20首李白经典诗作

## 🌟 在线访问

**网站地址：** https://lin-1259.github.io/libai-poetry/

## ✨ 特性

- 🎨 **渐变背景** - 紫蓝色渐变，自动循环动画
- 🌸 **飘落花瓣** - 20个半透明花瓣飘落特效
- 💎 **毛玻璃卡片** - 半透明背景，模糊效果
- ✨ **优雅动画** - 淡入、滑动等多种动画效果
- 📱 **响应式设计** - 手机、平板、电脑都完美显示
- ⌨️ **快捷键** - 按空格键快速换诗
- 🎲 **随机展示** - 每次打开或刷新随机显示一首

## 📖 诗词来源

数据来源：[chinese-poetry/chinese-poetry](https://github.com/chinese-poetry/chinese-poetry) 开源项目

- 橫吹曲辭·關山月
- 相和歌辭·對酒
- 等20首...

## 🚀 使用方法

1. 访问网站：https://lin-1259.github.io/libai-poetry/
2. 点击"换一首"按钮或按空格键切换诗词
3. 享受唯美的诗词展示
